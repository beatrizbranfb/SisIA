# Sobre o repositório 📁

- Este repositório está sendo utilizado para o desenvolvimento de um agente de IA, que tem como funcionalidade auxiliar nos fluxos do sistema utilizado pelo Tribunal de Contas da União.
- O agente aqui desenvolvido, é similar ao que está sendo feito, pelos mesmos participantes que fazem parte do repositório, para o TCU.

⚠️ O desenvolvimento aqui feito serve para registro de atividades individuais, e treinamento pessoal relacionado a LLM com RAGs e Finetuning, não sendo o utilizado oficialmente pelo TCU.
- O original está sendo desenvolvido externamente, não possuindo registro aqui no GitHub
